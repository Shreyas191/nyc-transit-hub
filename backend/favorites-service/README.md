Favorites Service

This microservice allows user to save and retrieve favorite subway, LIRR, and MNR stations or routes and provides real time arrival and departure updates.

Features
	•	Add, update, delete, and retrieve favorites.
	•	Fetch real-time updates for subway, LIRR, and MNR feeds.
	•	Returns human-readable timestamps.

Setup Instructions:
1. Clone the repository
2. Setup virtual environment
3. Install dependencies
4. Run the service - It will start on http://127.0.0.1:8003/

•	Need to update auth.py with real Firebase authentication.
